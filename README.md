# musicplayer
在线音乐播放器 - Web Music Player

项目简介

这是一个基于纯前端技术（HTML5、CSS3、JavaScript）构建的在线音乐播放器，专为静态托管设计。播放器无需后端服务器支持，可以轻松部署在 GitHub Pages 或其他静态托管服务上。

创建日期：2026年2月5日
最后更新：2026年2月5日

功能特性

🎵 核心播放功能

· 音乐播放控制：播放/暂停、上一首/下一首切换
· 进度控制：可视化进度条拖拽，实时显示播放时间
· 音量调节：滑块控制音量大小
· 播放列表：显示所有歌曲，点击直接切换
· 键盘快捷键：空格键播放/暂停，左右箭头切换歌曲
· 自动播放下一首：歌曲结束后自动播放下一首

🎨 用户界面

· 响应式设计：完美适配桌面和移动设备
· 现代化UI：渐变背景、圆角设计、平滑动画
· 专辑封面显示：每首歌曲显示对应的封面图片
· 播放状态指示：高亮显示当前播放歌曲
· 用户友好提示：清晰的操作指引和状态反馈

🔧 技术特点

· 纯静态实现：100%前端技术，无需后端
· 轻量级：无外部框架依赖，加载速度快
· 可定制性强：代码结构清晰，易于修改和扩展
· GitHub Pages友好：完美支持静态托管
· 跨平台兼容：支持所有现代浏览器

快速开始

1. 本地使用

1. 下载或克隆项目文件
2. 将您的MP3音乐文件放入music文件夹（重命名为song1.mp3、song2.mp3、song3.mp3）
3. 双击index.html在浏览器中打开
4. 如需更换图片，修改script.js中的图片链接

2. 部署到GitHub Pages

1. 创建GitHub仓库：username.github.io
2. 将所有文件上传到仓库根目录
3. 进入仓库设置 → Pages → 选择main分支 → 保存
4. 等待几分钟后访问：https://username.github.io

3. 自定义配置

· 修改播放列表：编辑script.js中的playlist数组
· 更改样式：编辑style.css文件
· 添加更多歌曲：在music文件夹添加MP3文件，在playlist数组中添加对应条目
· 更换专辑封面：修改图片链接或使用本地图片

技术栈

· HTML5：页面结构，使用Audio API播放音乐
· CSS3：响应式布局，现代化UI设计
· JavaScript (ES6)：播放器逻辑和交互
· Font Awesome：图标库
· Unsplash：免费高质量图片（可选）

使用说明

1. 播放/暂停：点击播放按钮或按空格键
2. 切换歌曲：点击播放列表中的歌曲或使用上一首/下一首按钮
3. 调节进度：拖拽进度条到任意位置
4. 调节音量：拖动音量滑块
5. 键盘控制：
   · 空格键：播放/暂停
   · 左箭头：上一首
   · 右箭头：下一首
